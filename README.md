# WemaBank Website Clone

A responsive clone of the WemaBank website built as a Level 1 school project at SQI to demonstrate HTML, CSS, and Bootstrap skills.

## 🎯 Project Overview

This project showcases web development fundamentals by recreating a banking website interface. It's designed to be responsive and user-friendly across different devices.

## 🛠️ Technologies Used

- **HTML5** - Semantic markup and structure
- **CSS3** - Styling and custom design
- **Bootstrap 5** - Responsive layout and pre-built components

## ✨ Features

- Responsive design that works on desktop, tablet, and mobile devices
- Clean and professional UI/UX
- Navigation menu with smooth interactions
- Banking services showcase
- Contact and information sections
- Bootstrap components for consistency
